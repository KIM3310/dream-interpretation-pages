Privacy

입력한 꿈 내용은 응답 생성을 위해 외부 API로 전달될 수 있습니다.

이 MVP는 별도 회원가입과 데이터베이스를 두지 않지만, 사용자가 입력한 꿈 설명은 OpenAI API 호출을 위해 서버 함수에서 처리됩니다. 민감한 개인정보나 타인의 실명, 연락처, 계정 정보는 입력하지 않는 편이 안전합니다.

수집 항목

  • 사용자가 직접 입력한 꿈 내용
  • 선택한 감정 상태와 꿈 직전 상황
  • 기본적인 서비스 운영 로그가 있을 수 있음

이용 목적

  • 꿈해몽 결과 생성
  • 오류 대응 및 남용 방지
  • 서비스 품질 개선

외부 처리자

OpenAI API가 응답 생성에 사용됩니다. Cloudflare Pages Functions는 요청을 수신하고 API를 중계하는 런타임입니다. 실제 운영 시에는 사용 중인 분석 도구와 광고 네트워크도 이 문서에 추가로 명시해야 합니다.

운영 전 체크

  • 실제 도메인, 운영자 이름, 문의 이메일을 반영하세요.
  • Google AdSense, Analytics 등을 붙이면 해당 쿠키/추적 고지를 추가하세요.
  • 법적 문구가 필요한 국가/지역이면 현지 기준에 맞춰 보강하세요.

이 문서는 MVP 초안입니다. 실제 공개 전에는 운영 국가 기준에 맞는 문구로 보강해야 합니다.
